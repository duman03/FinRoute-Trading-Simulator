# FinRoute

FinRoute, teknik analiz ve mum grafikleri (candlestick) öğrenmek isteyen kullanıcılar için tasarlanmış eğitici bir borsa ve grafik okuma simülasyon oyunudur.

## Özellikler

- Modern, **dark theme** arayüz
- Sanal bakiye ve dinamik portföy takibi
- Mock fiyat verileriyle güncellenen grafik ekranı
- Teknik analiz odaklı rastgele eğitim kartları
- Basit **Al / Sat** mekanizması ile sanal işlem

## Mottomuz

> Ezbere işlem yapmayı bırak. Kendi rotanı çiz.

## Çalıştırma

1. Bağımlılıkları yükleyin:

```bash
npm install
```

2. Expo geliştirme sunucusunu başlatın:

```bash
npm start
```

3. Expo Go ile mobil cihazınızda veya emülatörde açın.
